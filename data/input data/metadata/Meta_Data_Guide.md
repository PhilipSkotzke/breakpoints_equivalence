# Meta Data Guide

This study uses data generated as part of a different project. As part of that project, the graded exercise test was conducted to establish individual intensity zones. The data set includes tests of 12 national-level male triathletes. The copy of this data used in this project was made on 2025-10-09.

The data file *data_NIRS_BPs_2025.xlsx* is stored at `data/input data` and contains the recordings of participant characteristics as well as information on ventilatory thresholds and NIRS breakpoints.

The codebook is available under `data/input data/metadata`.

**Data Sharing**

Data users are kindly requested to notify the original authors if they intend to use the data for a new publication or significant analysis. Should the new analysis represent a substantial extension or re-analysis requiring the original authors' expertise or direct input, an offer of co-authorship to the original authors is recommended as long as the original authors meet the guidelines of the ICMJE (International Committee of Medical Journal Editors) guidelines.
